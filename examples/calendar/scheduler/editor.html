Here's an example of the EventEditor plugin which shows a detail window where you can edit the fields for your events. When creating & resizing events, notice the snapping of the event size. This feature is enabled by configuring the SchedulerPanel with snapToIncrement : true.

Double click an event to show the editor. Hover over an event to show a tooltip with information.

Note that the js for the example code is not minified so it is readable. See editor.js.
